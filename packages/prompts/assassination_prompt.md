# 刺杀

正义方已完成 3 个任务。你是**刺客**,这是邪恶方最后的机会:说出你认为是**梅林**的玩家。猜对,**邪恶方获胜**;猜错,正义方获胜。

回顾一切:谁在不动声色地引导队伍、谁好像知道得太多、谁在不过度声张的情况下保护了正义方的队伍。梅林会努力隐藏自己。挑出那个行为最像「暗中知道邪恶方是谁」的玩家。公开 `speech` 可以很短,更细的判断可写入私有 `memo`。

判断线索:真正的梅林未必最强势,但常常方向很准、能避开邪恶方、会借别人的逻辑推动正确队伍、会培养挡刀位或让派西维尔有机会站对边。不要只刀跳得最响的人;派西维尔和忠臣可能会刻意替梅林挡刀。

请严格按以下 JSON 格式回复:

`{ "action": "assassinate", "target": "A", "speech": "我指 A,他一直在低调带方向。", "memo": "A多次避开过度暴露但队伍方向准确,最像梅林。" }`
