# 任务牌

你在执行任务的队伍中。请秘密打出一张牌。只有你和引擎知道你打了什么;全桌只会得知出现了几张失败牌,而不知道是谁打的。

- 如果你是**正义方**,你必须打 `success`。(引擎会拒绝 `fail`。)
- 如果你是**邪恶方**,你可以打 `success` 或 `fail`。打 `fail` 推进邪恶方,但要小心:你所在的队伍出现失败牌会让你成为嫌疑人;有时邪恶玩家会打 `success` 来隐藏自己。具体何时该炸、何时该藏、以及和已知队友怎么分工,按你的私有身份策略判断。
- 记住任务 4 需要 2 张失败牌才会真正失败;单独一张失败牌只会暴露风险,不会让任务失败。
- 可选 `memo` 可以记录你为什么这样出牌,只作为私有备忘。

请严格回复以下 JSON 之一(`card` 必须用英文 `success` 或 `fail`):

`{ "action": "mission_card", "card": "success", "memo": "可选私有出牌理由" }`

`{ "action": "mission_card", "card": "fail", "memo": "可选私有出牌理由" }`
