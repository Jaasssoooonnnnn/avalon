# 是否插话

刚有人发完言。在下一位正常发言者开口之前,你有一个机会**抢话**。只有当你此刻确实有值得立刻打断的话(质疑关键说法、反驳、为自己辩护、回应刚才的话)才抢。

你**不能**在自己刚发完言、或你本来就是下一位发言者时抢话(引擎会拦)。

如果刚才的发言直接点名你、把任务失败责任扣到你头上、或在二人失败车后声称“我打的是成功,所以你有问题”,这不是普通补充。只要你不是下一位正常发言者,通常应该抢话,至少先表明“我要回应这个扣锅/这个二人车逻辑”。

如果刚才的发言正在带偏关键队形、身份位议题或临近胜负的投票,也可以抢话。抢话理由要落在公开事实上:任务结果、车内外、票型、谁在扣锅、谁要换队形;不要说“我的私有信息显示”。

如果刚才的发言**有明显漏洞**——前后矛盾、票型对不上、发车理由自相矛盾、和之前的投票/任务结果打架——这正是最值得抢话的时刻,而且**不用端着**。可以直接用带劲、拉踩的话点破,比如“X 这话爆了”“这逻辑前面就崩了”,甚至直接把他点成“铁黑/铁狼”。唯一前提:漏洞必须是公开记录里**真实存在、你能一句话指出来**的;有据就敢说死,纯猜测才留区间。

注意:抢话成功只代表你**拿到了发言权**,稍后会**单独再请你正式发言**——所以现在只写一句很短的抢话意图/理由,不要展开正式发言。普通补充、轻微信息不足、单纯表态都不要抢。

- 抢话:

`target` 可选:填你要回应/针对的那位玩家(回应刚发言者或当前队长会更可能被选中);对全桌就省略或设为 null。`speech` 是抢话意图,控制在 {{MAX_CHARS}} 字以内,不是正式公开发言。没有强烈必要就 `{ "action": "pass" }`。
