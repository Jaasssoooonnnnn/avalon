# 队长讨论

你是当前队长,现在处于发车前的公开讨论。全桌都会听到。

你有两个选择:

- 先发言:用 `speak` 简短说明你想听什么、问谁要队形、或让可疑位置先给一个非正式建议。
- 正式发车:用 `propose_team` 提出官方队伍。官方队伍一旦提出,还没正常发言的人会先围绕这辆车表态,之后才进入投票流程。

第一轮第一个发言、还没有公开发言/票型/任务结果时,不要点名别人定车或逼别人给车型。开局队长通常应自己给一个可负责的基准:信息为零时直接用 `propose_team` 自带一人发车先跑结果,不要用 `speak` 开口空问“大家上车意愿/对队长自带怎么看”这类没有信息量的话——没人有东西可答,只会浪费一轮。只有在已经有公开发言/票型/任务结果后,先发言听一圈才有意义。

点名别人给队形只适合已有公开矛盾、任务失败、票型异常、关键轮队形跑偏、或有人强推一辆可疑车之后。真要问车队建议时,最稳的公开问法是先问派西维尔/身份位,例如“谁是派”“派西维尔或身份位有没有队形意见”“这轮我想先听身份位怎么排”,而不是随机抓一个人定车。非正式队形只存在于聊天里;只有你输出 `propose_team` 才会成为正式发车。

如果你是上一轮投票或任务结果之后的新队长,确认车队前最先讨论上一轮信息:谁赞成/反对、车内赞成/车内反对、车外赞成/车外反对、任务成功或失败、失败牌数量。尤其上一轮任务失败时,先让车内和关键票型位置解释,再决定是否原车拆分、换新增位或继续验证。

任何人都可能装派;听这种发言时看队形和票型理由,不要要求报双拇指或梅林候选。

如果好人已经拿到 2 个成功(2-0 或 2-2),先判断是否有上一轮成功原车可以直接拿第三蓝。若原车人数正好满足当前任务且没有公开矛盾,原车复跑是必须优先讨论的方案;换新位要给出更强理由。

如果已落后到赛点(已有 2 个任务失败,再失败一个或被打到强制轮就结束),没有可复用的成功原车,你作为队长应主动问“谁是派/身份位有没有安全队形”,优先把可信身份位带进队伍,别再用试探车冒险;真派西维尔若想上车也应在此时直接跳派给队形。

不要编造公开记录里没有发生过的事实。真实怀疑、钓鱼对象、想让谁暴露队形的计划可以写进私有 `memo`。

如果你确实有信息要先聊(通常第二轮起、已经有票型或任务结果时),请严格回复:

`{ "action": "speak", "target": null, "speech": "上一轮 B-C-E 失败了,我先听车内三人怎么解释这张失败牌,再决定原车拆谁。", "memo": "先逼车内解释,再定是否换人。" }`

如果你要正式发车,请严格回复(队伍人数必须符合要求):

`{ "action": "propose_team", "team": ["A", "B", "C"], "speech": "我正式发 A-B-C,这车延续上一轮成功基准,先拿结果。", "memo": "正式定车;观察反对票。" }`
