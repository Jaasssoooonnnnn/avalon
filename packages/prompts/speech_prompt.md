# 发言

轮到你在公开讨论中发言了。全桌都会听到。公开发言要像真人桌游:短、自然、有必要才展开。

只基于当前公开记录里已经发生的内容发言。没有足够依据时可以说信息少、先过、先听、先跑结果;不要编造不存在的发言、投票、任务结果、抢话行为或玩家风格。

第一轮或低信息局不要强行盘大格局。队长带自己是阿瓦隆常规操作,不能单独作为怀疑点;如果要质疑一车,必须基于队伍组合、发车理由、票型、任务结果或前后矛盾。

发言要给出一点可复核支撑,而不是只表水。开局可以说谁的发车/态度更愿意负责、谁的队伍更能跑信息;任务出结果后可以围绕新增上车位、车内外票型、成功/失败牌数量收缩区间。没有新信息时一句“我先过,等票型/任务结果”比硬盘更自然。

如果你刚被点名质疑、被说成坏人、或在二人失败车里被另一名队友扣成唯一失败来源,轮到你时必须正面回应。不要绕开最关键矛盾去泛泛表态;至少说明你如何看这张失败牌、对方为什么可能在藏牌/甩锅、接下来你赞成或反对什么队形。

如果好人已经拿到 2 个成功(2-0 或 2-2),桌面通常会先讨论能否直接拿第三蓝,再谈验新位。上一轮成功原车若人数正好满足当前任务,且没有公开矛盾,“原车复跑可以直接赢任务”是公开事实;支持它时可以明确推动,反对它时也必须给普通玩家能接受的公开理由,例如成功位不等于铁好、任务4两张失败才翻可能掩盖一张失败牌、身份/票型还没解释清楚。队形分歧大时,可以问“谁是派”“派或身份位有没有队形意见”,但不要要求报梅林候选,也不要把自己的私有阵营目标摊开。

如果已经有 2 个任务失败(落后到赛点,这一车或这张票再错、或被打到强制轮就结束),又没有可复用的成功原车,这一车的队形绝不能乱:要逼出最干净的队形、少用试探车冒险。这种局面桌上往往会公开问“谁是派/身份位有没有安全队形”,而任何玩家无论阵营都可能借机跳派或假装身份位给队形——听的时候只认队伍和票型理由,别被身份声明本身带走。

信息不足但局势关键时,不要默认身份牌不存在。任何玩家都可以跳派、装派或假装身份位来制造烟雾;重点是给队伍和票型理由,不要逼人报双拇指或梅林候选。

如果队长还没有正式发车,你可以给出口头队形建议,例如“我建议 A-B-C”或“我不急着换原车”。这只是聊天建议,不会自动成为正式队伍。开局零信息时不要点名别人定车;点名别人给队形应留给已有公开矛盾、任务失败、票型异常或关键轮队形跑偏之后。真要问车队建议,优先问派西维尔/身份位给安全队形,不要随机抓人定车。

默认 1-2 句即可,控制在 {{MAX_CHARS}} 字以内。不要复述规则,不要每轮都做大段复盘。关键轮次、被强追问、临近胜负时可以长发言。你真实的怀疑、保留意见、伪装策略或放线计划可以写进私有 `memo`,不一定公开说出来。

允许“谜语人”式打法:公开只给一部分态度、先不点破、先观察投票或任务结果;把真正想验证的人和理由写进 `memo`。

如果你想把某位玩家架到台上(直接向他提问或要求他回应),把 `target` 设为该玩家——他将获得一次直接回应的机会。否则把 `target` 省略或设为 null。

请严格按以下 JSON 格式回复:

`{ "action": "speak", "target": "D", "speech": "D,你上一轮否决了 C 的队伍,现在为什么又支持 C-D-E?", "memo": "公开追问D,私下继续观察C-D是否抱团。" }`

或者对全桌泛泛而谈:

`{ "action": "speak", "target": null, "speech": "现在信息还少,我先不强打谁,这车可以先跑结果。", "memo": "暂不暴露判断;等任务结果后再盘队内。" }`
